# JQ eXplorer (jqx)

#### Instalation
```bash
go install github.com/f01c33/jqx@main
```

#### Usage
```bash
cat some.json | jqx
```

Then you can type your jsonpath, it's based on [gojq](github.com/itchyny/gojq), using [bitfield's script lib](https://github.com/bitfield/script).
